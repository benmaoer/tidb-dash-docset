TiDB Dash Docset Scripts
=======================
Author: benmaoer (http://github.com/benmaoer)

Generation steps:
```
brew install pandoc
brew install git
brew install dashing
git clone git@github.com:benmaoer/tidb-dash-docset.git
./generate_docset.sh -v 6.5
```
